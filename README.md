# 🐻 The Poo Bears · Family Kitchen

> Meals so good they'll make you shit your pants.

A single-file family meal-planning app for the Poo Bears (Kodie, Lauren, Kaiden & Asher). Pick the week's meals, get a Whole Foods grocery list grouped by aisle, track the Tiffin Farm quarter cow, flag pantry items when you run out, and print clean recipe cards.

## What's in it

- **40 recipes** — Asian bowls & stir-fries, quarter-cow meals, quick weeknights, and a pile of viral hits, each with detailed steps, prep/cook times, nutrition, a kid note, pro tips, a "serve with" side, and a recommended sauce.
- **15 homemade sauces** — yum yum, bang bang, teriyaki, gochujang, chimichurri, comeback, marinara, and more — every recipe pairs to one.
- **The Plan** — add meals, then one tap builds a grocery list grouped by Whole Foods department, with tap-to-add links into the Whole Foods storefront.
- **The Freezer** — track the Tiffin Farm quarter-cow cuts (ground, ribeye, NY strip, brisket…).
- **Pantry** — a full spice/pantry library; tap an item when you run out and it lands on a restock list.
- **Save PDF** — every recipe and sauce prints a clean one-page card with the photo.

Everything saves locally (localStorage) — no accounts, no backend.

## Tech

Plain HTML/CSS/JS in one `index.html`. Plus Jakarta Sans, real food photography, an installable PWA manifest, and a social share card. Hosted on GitHub Pages.

## Local dev

Open `index.html` in a browser. Dev tooling (image generation, the Playwright click-test, brand-asset rendering) lives in `dev/`.
